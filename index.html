Mode d'emploi

Calendrier

  • Glissez à gauche/droite pour changer de mois
  • Cliquez sur le titre du mois pour afficher la vue annuelle
  • Cliquez sur une journée pour voir les réservations
  • Utilisez le bouton + pour créer une réservation
  • Le bouton Aujourd'hui apparaît quand vous quittez le mois courant

Réservations

  • Les réservations apparaissent comme des barres colorées continues
  • Cliquez sur une barre ou un jour pour voir le détail
  • Vous pouvez modifier ou supprimer une réservation depuis le détail du jour

Inventaire

  • Organisé en colonnes par catégorie
  • Utilisez les boutons Aucun / Un peu / Max pour indiquer les quantités
  • Ajoutez des articles avec le formulaire en bas

Notifications

  • Activez les notifications dans Paramètres
  • Recevez une alerte à chaque nouvelle réservation, modification ou suppression

Installation

Sur iPhone : Safari → Partager → Sur l'écran d'accueil.
Sur Android : Chrome → Menu → Installer l'application.
